# Gym Membership System (Java OOP Project)

## Description
This is a simple Java console-based Gym Membership System.  
The user enters their details and chooses a membership type (Basic, VIP, or Premium).

### Features
- Takes user input using Scanner
- Allows membership selection
- Displays membership result
- Uses simple OOP concepts
#### OOP Concepts Used

# Classes and Objects
- Member, VIPMember, and PremiumMember are classes
- Objects are created from these classes

# Encapsulation
- name is private
- Accessed using the getter method getName()

# Inheritance
- VIPMember and PremiumMember extend Member

# Polymorphism
- showPackage() is overridden in child classes

# Method Overloading
- showPackage() and showPackage(String message)

# Access Modifiers
- Uses private, protected, and public
